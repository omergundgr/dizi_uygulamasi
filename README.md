# Dizi Film İzleme Uygulaması

Flutter / Dart dili ile yazılan film izleme uygulaması m3u8 ve diğer video formatlarını destekler.
- Dizi ve Film Kategorileri
- Image Slider
- Arama Kutusu
- Yan Menü
- Firebase Firestore Veri Yapısı
